# TinyOS
A OS written by me in C.
It's really lightweight.
## Why?
i made it because i wanted to learn a bit of Bare Metal C or also called freestanding C that's why i took 5 hours of my lifetime and made this OS
## Infos
It has about 2500 Lines of code right now.
the filesystem is persistent.
you have to Emulate it with VGA(because it uses VGA Framebuffer, well now it kinda works on HDMI)
It only supports english keyboard layout
It doesn't work on USB drives because USB driver is missing.
SATA disks work.
