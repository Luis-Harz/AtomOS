What is TinyOS?

It is a selfmade Kernel in C. Right now it has 3700 lines of code.

Features

  • Own scheduler
  • Multitasking
  • VGA-Framebuffer*
  • Shell
  • Login
  • Filesystem
  • ATA

VGA-Framebuffer

It actually uses if available a Framebuffer set up by Grub.

It should also work on HDMI and DP, you can try it out.
